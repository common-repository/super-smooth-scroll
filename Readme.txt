Super Smooth Scroller will change your default lazy browser scrollbar to nice fancy and smooth. 

== Description ==

Very tiny, fast loaded this plugin is. Will add smooth scrolling expericence to your site which your visitor will like sure! So, just get it, install it and see the magic. 

working demo at --> http://plugins.wpflaty.com/super-smooth-scroll/

== Installation ==

1. Upload this plugin in your server ` /wp-content/plugins/ `
2. Activate the plugin through the 'Plugins' menu in WordPress
